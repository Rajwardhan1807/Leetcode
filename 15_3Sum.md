# 15. 3Sum

## Problem Statement

Given an integer array `nums`, return all the unique triplets `[nums[i], nums[j], nums[k]]` such that:

* `i != j`
* `i != k`
* `j != k`
* `nums[i] + nums[j] + nums[k] == 0`

The solution must not contain duplicate triplets.

---

## Example 1

```text
Input: nums = [-1,0,1,2,-1,-4]

Output: [[-1,-1,2],[-1,0,1]]
```

## Example 2

```text
Input: nums = [0,1,1]

Output: []
```

## Example 3

```text
Input: nums = [0,0,0]

Output: [[0,0,0]]
```

---

## Approach

### Sorting + Two Pointers

1. Sort the array.
2. Fix one element at a time.
3. Use two pointers:

   * `j` → starts after the fixed element.
   * `k` → starts from the end.
4. Calculate the sum:

   * If the sum is `0`, store the triplet.
   * If the sum is less than `0`, move `j` forward.
   * If the sum is greater than `0`, move `k` backward.
5. Skip duplicate elements to avoid repeated triplets.

---

## Why This Works

Sorting allows us to efficiently move the two pointers based on the current sum.

For example:

```text
[-4,-1,-1,0,1,2]
```

Fix:

```text
-1
```

Use two pointers:

```text
-1  -1   0   1   2
 ^             ^
```

Current sum:

```text
-1 + (-1) + 2 = 0
```

Store:

```text
[-1,-1,2]
```

Move both pointers while skipping duplicates.

---

## Code

```cpp
class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums) {

        vector<vector<int>> output;

        sort(nums.begin(), nums.end());

        for(int i = 0; i < nums.size(); i++) {

            if(i > 0 && nums[i] == nums[i - 1])
                continue;

            int j = i + 1;
            int k = nums.size() - 1;

            while(j < k) {

                int sum = nums[i] + nums[j] + nums[k];

                if(sum == 0) {

                    output.push_back({nums[i], nums[j], nums[k]});

                    j++;
                    k--;

                    while(j < k && nums[j] == nums[j - 1]) j++;
                    while(j < k && nums[k] == nums[k + 1]) k--;

                }
                else if(sum < 0) {
                    j++;
                }
                else {
                    k--;
                }
            }
        }

        return output;
    }
};
```

## Dry Run

### Input

```text
nums = [-1,0,1,2,-1,-4]
```

After sorting:

```text
[-4,-1,-1,0,1,2]
```

Fix `-1`:

```text
-4 -1 -1 0 1 2
    i      j k
```

Sum:

```text
-1 + (-1) + 2 = 0
```

Triplet found:

```text
[-1,-1,2]
```

Move pointers.

Next sum:

```text
-1 + 0 + 1 = 0
```

Triplet found:

```text
[-1,0,1]
```

Final Answer:

```text
[[-1,-1,2],[-1,0,1]]
```

## Complexity Analysis

### Time Complexity

```text
O(N²)
```

* Sorting takes **O(N log N)**.
* The two-pointer search runs in **O(N²)** overall.

### Space Complexity

```text
O(1)
```

Ignoring the space used for the output.
